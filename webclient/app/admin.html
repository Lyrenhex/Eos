Eos Administrator Portal

Hi, welcome to the Eos Administrator Portal. This area is restricted to accounts which are registered as administrators, which must be set by the server syadmin. Accounts created through this page are not administrator accounts, and this page should not be used for account creation.

When moderating chat logs:

  1. Read the entire chat history, and take context into account. If someone says, "He's pissing me off", for example, this is NOT a violation of the Eos service terms, and nor should this be treated as such. Furthermore, we don't store which user reported the chat; please account for either user having been in the wrong.
  2. Once the chatlogs have been reviewed, please take one of two steps:
  3. If the chatlog violates the law, or you worry for a user's safety, or other serious concern:
    1. DO NOT submit a report decision. Instead, please press the "flag" button, which will ensure that the report is duplicated in a safer location to avoid accidental loss of the report.
    2. Contact Damian (Lyrenhex) either via Discord or by emailing him about the situation.
    3. Continue as normal. The situation is no longer under your responsibility. Thank you!
  4. Otherwise, if the chatlog does not violate the law:
    1. If a user is found to have violated the Eos terms of service or other corresponding rules, but not a criminal offence, then please copy the User ID of the offending user and enter this into the "Malign Party" input box at the bottom of the page. IF NO USER IS AT FAULT, DO NOT ENTER ANYTHING INTO THIS BOX.
    2. Press "Submit decision". The server will handle the rest.
